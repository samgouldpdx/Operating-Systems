One operating system function is the ability to run applications.
It is the ability to run an application such as Word processor by locating it and loading it into the primary memory. 
Most operating systems can multitask by running many applications at once.

The operating system manages the Primary Memory or Main Memory.
Main memory is made up of a large array of bytes or words where each byte or word is assigned a certain address.
Main memory is a fast storage and it can be accessed directly by the CPU.
